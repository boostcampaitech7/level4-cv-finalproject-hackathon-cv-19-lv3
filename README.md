# 🔨 딴 - 따라 Service Architecture

<h3>1. SNS(Youtube, TikTok, Instagram)에서 원하는 숏폼 챌린지를 골라보세요!</h3>

- 숏폼을 즐기다 챌린지하길 원하는 영상이 보인다면 링크만 가져와보세요.

<br/>

<br/>
<h3>2. 노래에 맞춰 신나게 딴따라 해보세요! </h3>

<br/>
<h3>3. 딴따라가 제공한 댄스 피드백으로 자신의 안무를 다듬어 보세요!</h3>
